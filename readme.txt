Houd op eenvoudige manier voorraad en verkoop bij.

== Description ==
In dit systeem kan men eenvoudig een hun voorraad plaatsen en bij iedere verkoop word de voorraad bijgewerkt.

Dit systeem is puur bedoeld voor de particulier die op eenvoudige wijze hun handel in de gaten wil houden.

Ordered list:

* Voorraad beheer
* Verkoop beheer
* Verkoop statistieken


== Installation ==
1. Installeer de map in "/wp-content/plugins/".
1. Activeer de plugin bij "Plugins" in je WordPress installatie.
1. Je kan gebruik maken van de plugin.
